# Icon Sets

Iconx provides access to 200,000+ icons from 150+ icon sets available in the
Iconify ecosystem. This page covers the most popular icon sets and their
characteristics.

## Popular Icon Sets

### Lucide (`lucide`)

**Description:** Modern, clean icons with consistent styling  
**License:** MIT  
**Icons:** 1,000+  
**Style:** Outline, 24px grid  
**Best for:** General UI, modern applications

```bash
iconx add --set lucide home user settings mail search edit trash plus minus
```

**Examples:**

- `home` - House icon
- `user` - User profile icon
- `settings` - Gear/cog icon
- `mail` - Email icon
- `search` - Magnifying glass
- `edit` - Pencil icon
- `trash` - Trash can icon

### Heroicons (`heroicons`)

**Description:** Tailwind CSS icon set  
**License:** MIT  
**Icons:** 230+  
**Style:** Outline and solid variants, 24px grid  
**Best for:** Tailwind CSS projects, clean interfaces

```bash
iconx add --set heroicons home user cog envelope magnifying-glass
```

**Examples:**

- `home` - House icon
- `user` - User icon
- `cog` - Settings icon
- `envelope` - Email icon
- `magnifying-glass` - Search icon

## Choosing the Right Icon Set

### Consider Your Design System

**Modern/Clean:** Lucide, Heroicons, Feather  
**Material Design:** Material Design Icons  
**Enterprise:** Carbon, Fluent UI  
**Established:** Font Awesome, Tabler

### License Considerations

Most icon sets use permissive licenses:

- **MIT:** Lucide, Heroicons, Tabler, Feather, Phosphor
- **Apache 2.0:** Material Design Icons, Carbon, Fluent UI
- **CC BY 4.0:** Font Awesome

### Icon Availability

Check icon availability before choosing:

1. Visit [Iconify Icon Sets](https://icon-sets.iconify.design/)
2. Browse your preferred icon set
3. Verify the icons you need are available
4. Note the exact icon names

## Using Multiple Icon Sets

You can mix icon sets in the same project:

```bash
# UI icons from Lucide
iconx add --set lucide home user settings mail search

# Brand icons from Font Awesome
iconx add --set fa-brands github twitter linkedin

# Specialized icons from Octicons
iconx add --set octicons repo issue pull-request
```

## Best Practices

### 1. Stick to One Primary Set

```bash
# Good: Consistent icon set
iconx add --set lucide home user settings mail search

# Avoid: Mixing styles unnecessarily
iconx add --set lucide home
iconx add --set heroicons user
iconx add --set feather settings
```

### 2. Use Aliases for Consistency

```json
{
  "aliases": {
    "house": "home",
    "cog": "settings",
    "envelope": "mail"
  }
}
```

### 3. Group Related Icons

```bash
# Navigation icons
iconx add --set lucide home dashboard settings help

# Action icons
iconx add --set lucide plus minus edit trash save

# Social icons
iconx add --set fa-brands github twitter linkedin
```

### 4. Preview Icons Before Adding

Always preview icons at
[icon-sets.iconify.design](https://icon-sets.iconify.design/) to ensure they
match your design needs.

## Finding Icons

### Search Methods

1. **Browse by Set:**
   [icon-sets.iconify.design](https://icon-sets.iconify.design/)
2. **Search by Name:** Use the search function on the Iconify website
3. **Category Browse:** Browse by categories like "arrows", "communication",
   etc.

### Icon Search Tips

- Search for common terms: "home", "user", "settings"
- Try variations: "house" vs "home", "cog" vs "settings"
- Check multiple sets for the same concept
- Look for filled vs outline variants
