## Introduction

Saas UI is an open-source Design System for building modern web application that have a native and high-quality feel to them.
Inspired by MacOS and tools like Linear, Attio and Superhuman, it is built with the goal of giving you easy to use primitives
to build intuitive and beautiful user interfaces without sacrificing development velocity.

Saas UI can be installed as a library using NPM or ejected into your own project for full control and ownership.
