## Procedures

The auth router has the following procedures:

### Get the current user

Returns the current logged in user and the workspaces they have access to.

- Access: `@protected`

```typescript
// Client components
const { data } = api.auth.me.useQuery()

// Server components
const data = await api.auth.me()
```
