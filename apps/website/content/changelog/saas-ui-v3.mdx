We're excited to announce the release of Saas UI v3! This release brings several new features and improvements to the framework.

## What's new in Saas UI v3?

### Chakra UI v3 Support

Saas UI v3 now supports Chakra UI v3. This means you can now use the latest version of Chakra UI with Saas UI.

### Improved Starter Kits

The Saas UI Next.js starter kit has been improved with several new features and improvements.
