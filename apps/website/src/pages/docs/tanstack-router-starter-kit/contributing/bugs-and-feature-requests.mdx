## Reporting bugs

When you find a bug or issue with the TanStack Router starter kit, please report it to the [GitHub repository](https://github.com/saas-js/tanstack-router-starter-kit/issues).

Please include as much information as possible, including:

- The version of the TanStack Router you are using.
- The browser and operating system you are using.
- A code example that reproduces the issue.

## Feature requests

We really value your feedback and ideas. You can submit features requests [on our roadmap](https://roadmap.saas-ui.dev).
Also feel free to start a discussion on Discord before submitting a feature request.
