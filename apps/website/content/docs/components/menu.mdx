## Anatomy

```jsx
import { Menu } from '@saas-ui/react/menu'
```

```jsx
<Menu.Root>
  <Menu.Button />
  <Menu.Content>
    <Menu.Item value="item" />
  </Menu.Content>
</Menu.Root>
```
