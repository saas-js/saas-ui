## Anatomy

```jsx
import { Steps } from '@saas-ui/react/steps'
```

```jsx
<Steps.Root>
  <Steps.List>
    <Steps.Item />
  </Steps.List>
  <Steps.Content />
  <Steps.CompleteContent />
  <Steps.PrevTrigger />
  <Steps.NextTrigger />
</Steps.Root>
```
