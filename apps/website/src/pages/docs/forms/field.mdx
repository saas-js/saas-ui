Field is a complete functional form control that renders a specific field type based on the data type used.
It handles the field state, renders a label, the input, help text, error messages

<ComponentLinks
  github={{ package: 'forms' }}
  npm={{ package: '@saas-ui/forms' }}
/>

## Import

- `Field`: The wrapper component that controls context, state and manages rendering field types.

```ts
import {
  Field,
  InputField,
  TextareaField,
  NumberField,
  SelectField,
  RadioField,
  CheckboxField,
  SwitchField,
  PasswordField,
  PinField,
  NativeSelectField,
} from '@saas-ui/react'
```

## Usage

The `Field` is a controlled component and should aways be wrapped with `Form`, as it depends on the form context.

## Build in field types

Build in field types are `text`, `textarea`, `number`, `password`, `select`, `native-select`, `radio`, `checkbox`, `switch`, `pin`.
Any other type will fallback to a regular HTML `input` element. Eg: `email` will render as a `InputField` with the type set to email.

### Text field

```jsx
<Form>
  <FormLayout>
    <Field type="text" name="name" label="Name" />
    {/* or: <InputField name="title" label="Title" /> */}

    <Field type="email" name="email" label="Email" />
    {/* or: <InputField name="title" type="email" label="Title" /> */}
  </FormLayout>
</Form>
```

### Textarea field

```jsx
<Form>
  <Field type="textarea" name="description" label="Description" />
  {/* or: <TextareaField name="description" label="Description" /> */}
</Form>
```

### Number field

```jsx

```

### Password field

```jsx
<Form>
  <Field type="password" name="password" label="Password" />
  {/* or: <PasswordField name="password" label="Password" /> */}

```

### Select field

```jsx
<Form>
  <Field
    type="select"
    name="role"
    label="Role"
    options={[
      { value: 'Developer' },
      { value: 'Designer' },
      { value: 'Management' },
    ]}
    value="developer"
  />
  {/* or: <SelectField name="role" label="Role" options={[{value: 'Developer'}, {value: 'Designer'}, {value: 'Management'}]}  /> */}
</Form>
```

## Advanced

### Creating your own field types.

```jsx live=false
import { registerFieldType } from '@saas-ui/forms'

import ReactSelect from 'react-select'

const ReactSelectField = registerFieldType('react-select', ReactSelect, {
  isControlled: true,
})
```

### Overriding default fields.

You can override any of the build in types by registering a field on the corresponding key.

```jsx live=false
import { registerFieldType } from '@saas-ui/forms'

import ReactSelect from 'react-select'

const ReactSelectField = registerFieldType('select', ReactSelect, {
  isControlled: true,
})
```
