---
title: Deploy to Netlify
description: This guide will show you how to deploy to Netlify
---

Follow these steps to deploy the monorepo to Netlify.

1. Create a new project on Netlify.
2. Connect your GitHub account.
3. Find the repository you want to deploy and click `import`.
4. Override the build command with;

```bash
build build:web
```

5. Start command

```bash
yarn start
```

6. Make sure to add the required environment variables.

See [environment variables](/docs/tanstack-router-starter-kit/configuration/environment-variables) for more information.

<Info>Always generate new secrets when deploying to production.</Info>
