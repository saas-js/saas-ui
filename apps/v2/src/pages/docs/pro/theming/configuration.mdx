---
title: Setting the Saas UI Pro theme
description: How to setup the Saas UI Pro theme in your project.
---

After intalling Saas UI Pro into your project follow these steps to make sure all Pro components are styled correctly.

```ts
import { extendTheme } from '@chakra-ui/react'

// 1. Import the pro theme
import { theme as baseTheme } from '@saas-ui-pro/react'

// 2. Extend your theme
const theme = extendTheme(
  {
    // your custom theme
  },
  baseTheme
)

// 3. Pass the custom theme to SaasProvider
function App() {
  return <SaasProvider theme={theme}>{children}</SaasProvider>
}
```
