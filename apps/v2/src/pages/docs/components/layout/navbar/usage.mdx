## Import

- `Navbar`: The container component that manages composition.
- `NavbarBrand`: The component used for branding.
- `NavbarContent`: The component that contains navbar items.
- `NavbarItem`: A navbar item.
- `NavbarLink`: A navbar link.

```ts
import {
  Navbar,
  NavbarBrand,
  NavbarContent,
  NavbarItem,
  NavbarLink,
} from '@saas-ui/react'
```

### Hide on scroll

Use the `parentRef` prop to specify the parent scroll container, defaults to the `window` object.

  Button,
  Box,
  Container,
  Stack,
  Skeleton,
  SkeletonText,
} from '@chakra-ui/react'

  return (
    <Box
      ref={parentRef}
      height="400px"
      overflowY="auto"
      position="relative"

      <AppShell
        navbar={
