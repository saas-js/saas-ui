Saas UI Pro is a frontend starterkit for SaaS projects and includes advanced and pre-built components and tools for billing, onboarding, feature management, sign-up flows and much more.
It comes with a boilerplate app that runs on NextJS and Electron.

Filters, Page, Toolbar, Filters, BulkActions, Kanban Board, DataGrid, Onboarding, Charts, Timeline, FeatureFlags, Editor.

### Pre-build templates and components

Authentication, Lemonsqueezy integration, Navigation, Dashboard, ListPage, SplitPage, Onboarding Flow, Contacts Module, Settings Pages, Billing Plans, User Management, User Profile Settings, Help Center.
